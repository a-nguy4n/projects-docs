About

Weather Wardrobe is a personalized IoT wardrobe assistant that helps users choose outfits based on live environmental data and weather forecasts. Built as a full-stack web app, it integrates ESP32 sensors, a secure user dashboard, and an AI assistant to deliver smart wardrobe recommendations.

After signing up, users can register their own ESP32 devices and access a secure dashboard that visualizes temperature data in real time via MQTT. The system combines sensor readings with weather forecasts to power an AI assistant that answers questions like “What should I wear today?”.
